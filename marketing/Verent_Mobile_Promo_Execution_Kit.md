## 1) iPhone Capture Shot List (Vertical 9:16)

Record every shot in portrait mode, 1080x1920, and keep fingers steady.
For each shot, capture:
- Take 1 at normal speed
- Take 2 at slightly slower gestures
- Take 3 backup with cleaner scroll/tap timing

### Capture Settings
- Do Not Disturb: ON
- Battery + signal icons clean
- Brightness: 70-80%
- Text size: default (do not zoom)
- Screen recording: ON, microphone OFF
- Keep each clip 4-8 seconds max

### Renter Flow Shots
1. App open -> renter dashboard visible.
2. Tap Explore -> scroll premium listings.
3. Open one listing details page.
4. Show booking/rental action tap.

### Lister Flow Shots

10. Show confirmation page + refreshed status.

### Wallet + Trust Shots

14. Show transfer summary card.

## 3) Canva Master Template Setup (Do Once)

Create a single Canva project named:
"Verent Promo Master - Mobile 9x16"

### Canvas
- Size: 1080 x 1920
- FPS target: 30
- Safe margin: keep text inside middle 80% width

### Brand Style
- Primary text: white or near-black depending on background
- Accent color: Verent green
- Secondary neutral: gray scale
- Font pairing:
  - Headline: Montserrat Bold
  - Body/captions: Open Sans
- Caption size:
  - Headline: 58-72
  - Body: 34-44
  - Labels: 24-28

### Reusable Components
1. Hook card (big 4-word statement)
2. Feature lower-third (icon + 3 words)
3. Status chip (Confirmed / Pending / Proof)
4. CTA end card
5. Split-role title card (Renter / Lister)

### Motion Rules
- Transition style: dissolve or slide only
- Transition duration: 0.2-0.4s
- Max one animation per scene
- No heavy zoom spam
