# Canva Master Checklist (Verent 9:16)

Use this checklist once, then duplicate the project for each video cut.

## Project Setup
- Create project: `Verent Promo Master - Mobile 9x16`
- Canvas size: `1080 x 1920`
- Frame rate target: `30 FPS`
- Add guides: keep all text inside center safe area

## Brand Layer
- Headline font: Montserrat Bold (fallback Poppins Bold)
- Body font: Open Sans (fallback Arial)
- Color set:
  - Primary dark: #0A0A0A
  - Primary light: #FFFFFF
  - Accent: Verent green
  - Neutral grays for chips and labels

## Reusable Components to Build
1. Hook card (big statement, max 5 words)
2. Split-role card (`For Renters` / `For Listers`)
3. Wallet trust card (`Confirmed` / `On-chain proof`)
4. CTA card with URL/contact
5. Lower-third banner (`Feature + icon`)

## Motion Settings
- Default transition: Dissolve
- Transition duration: 0.2-0.4 sec
- Text animation: Rise or Fade only
- Avoid heavy zoom or multi-effect stacks

## Audio Setup
- Track 1: Voiceover (priority)
- Track 2: Music bed at low volume
- Track 3: SFX for taps/swipes only where needed

## Output Presets
- Preset A: `verent_social_v1.mp4` (60s)
- Preset B: `verent_investor_v1.mp4` (90s)
- Quality: highest, MP4, 1080x1920
